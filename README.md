# Old-Ben

A slackbot with the personality of everyones favorite old Jedi Master Obi-Wan Kenobi. Powered by Node.js, Slack's Bolt API, and the Force.

## Loves

- warm greetings
- dad jokes
- retro diners
- the Jedi Order

## Hates

- absolutes
- trade disputes
- the Sith

## Tech Used

- Node.js
- Slack Bolt API

## Dependencies

- @slack/bolt
- dotenv
- node-fetch
